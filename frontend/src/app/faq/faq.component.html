Perguntas Frequentes

Siscarb é uma plataforma criada para auxiliar mães de crianças moradores do município de São Paulo portadoras de diabetes tipo 1 a calcular a quantidade de carboidratos ingeridos em uma refeição.

Para consegui registrar o seu histórico é necessário criar uma conta e estar logado. Após isso, basta clicar no botão do carrinho no topo da página e em "Salvar Histórico" que os cálculos serão salvos.

O cardápio pode ser acessado via menu principal. Será carregado um frame com um mapa onde poderá buscar pelo nome da escola onde a criança frequenta. Após selecionar a escola e a data, será carregado o cardápio do dia. Clique sobre a idade para ver o cardápio.

Sim, o Siscarb é software livre e está disponível no Github. Você pode acessar o código fonte clicando no ícone do Github no rodapé da página.
